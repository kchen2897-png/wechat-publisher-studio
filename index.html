怎么用

  1. 标题填在"文章标题"，正文代码粘到"代码 / 正文"。
  2. 如果内容里既有 HTML 又有 Markdown，选择"智能混合"。
  3. 点击"转换预览"，确认右侧效果。
  4. 点击"复制到公众号"，到微信公众平台正文编辑区按 Ctrl + V。

注意：公众号封面图仍需在后台单独上传；正文里的本地图片路径通常无法直接带过去。
