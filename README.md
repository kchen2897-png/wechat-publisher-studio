# 公众号富文本转换器

本地访问地址：

```text
http://127.0.0.1:8787/
```

GitHub Pages 发布后访问地址：

```text
https://kchen2897-png.github.io/wechat-publisher-studio/
```

用途：

- 把 HTML / CSS 代码转换成微信公众平台可粘贴的富文本。
- 把 Markdown 正文转换成公众号排版。
- 把 HTML 和 Markdown 混合内容智能转换成公众号富文本。
- 把普通文本自动分段并生成预览。
- 单独填写和复制文章标题，也可以选择把标题显示在正文顶部。
- 一键复制渲染后的正文，而不是复制代码本身。

推荐流程：

1. 在“文章标题”里填写标题。
2. 选择输入类型，混合内容推荐选择“智能混合：HTML + Markdown”。
3. 粘贴 HTML / CSS / Markdown / 普通文本。
4. 点击“转换预览”。
5. 点击“复制标题”，粘贴到公众号标题栏。
6. 点击“复制到公众号”，到正文编辑区按 `Ctrl + V`。
7. 在公众号后台手机预览，确认样式后发布。

提示：

- 带 `<style>` 的 HTML 会尽量转换成内联样式。
- `<script>`、`iframe`、事件属性等会默认清理。
- 本地图片路径通常不能直接复制到公众号后台，正文图片建议在公众号编辑器内重新上传。
- 公众号平台会二次清洗样式，最终效果请以公众号后台预览为准。
